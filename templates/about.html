Tentang BookFinder

BookFinder adalah aplikasi web yang menggabungkan pencarian buku, crawling Wikipedia, analisis NLP, dashboard visual, serta otomasi data cuaca dalam satu sistem terpadu.

Fitur Utama

  • Pencarian buku via Google Books API
  • Crawling Wikipedia
  • Analisis NLP
  • Dashboard grafik Chart.js
  • Wordcloud otomatis
  • Otomasi cuaca (CSV & PNG)
